* Git Gutter Fringe
https://github.com/emacsorphanage/git-gutter-fringe

#+begin_src emacs-lisp
(use-package git-gutter-fringe
  :ensure t
  :config
  (global-git-gutter-mode t)
  (define-fringe-bitmap 'git-gutter-fr:added [224] nil nil '(center repeated))
  (define-fringe-bitmap 'git-gutter-fr:modified [224] nil nil '(center repeated))
  (define-fringe-bitmap 'git-gutter-fr:deleted [224] nil nil '(center repeated))
  (custom-set-faces
   '(git-gutter-fr:added    ((t :foreground "#4a7d56" :background unspecified)))
   '(git-gutter-fr:modified ((t :foreground "#4c4f7d" :background unspecified)))
   '(git-gutter-fr:deleted  ((t :foreground "#7d4c52" :background unspecified)))))
#+end_src

* Git Time Machine
[[https://github.com/emacsmirror/git-timemachine][git-timemachine]] is a program that allows you to move backwards and forwards through a file's commits.  'SPC g t' will open the time machine on a file if it is in a git repo.  Then, while in normal mode, you can use 'CTRL-j' and 'CTRL-k' to move backwards and forwards through the commits.

#+begin_src emacs-lisp
(use-package git-timemachine
  :ensure t)
#+end_src

* Magit
[[https://magit.vc/manual/][Magit]] is a full-featured git client for Emacs.

** Magit Todos
[[https://github.com/alphapapa/magit-todos][magit-todos]] displays TODO/FIXME/etc items in the Magit status buffer.

#+begin_src emacs-lisp
(use-package magit-todos
  :ensure t
  :after magit
  :config
  (magit-todos-mode 1))
#+end_src
